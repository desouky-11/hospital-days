# برنامج لحساب عدد ايام جلسات الغسيل في المستشفى

هذا برنامج بال HTML, CSS, JavaScript لحساب عدد ايام جلسات الغسيل بالمستفى بمرضى الغسيل الكلوي

## التعديل الأول

اضف المجموعة الأولى والثانية كخيار
وامامهم ايام الأسبوع ك تشيك بوكس

بحيث ان المستخدم يختار ايام الأسبوع لكل مجموعة اولا (واجعل الاختيار الافتراضي ان مجموعة 1 سبت واثنين واربع ومجموعة 2 احد ثلاثاء خميس

ثم يتم الحساب

ايضا اضف خيار حساب الأيام بشكل سنوي حيث يحسب لكافة شهور السنة واذا اخترنا هذا الاختيار يجعل اختيار الشهور غير مفعل

كما اضف خيار لحفظ السنة بالكامل كملف txt

## التعديل الثاني

 تطوير البرنامج لتكون طريقة الحساب
شهر بشهر
تقرير مرن يتم تحديد اول شهر للحساب و  آخر شهر للحساب
التقرير السنوي

وكل اختيار استطيع حفظ النتائج كملف نصي

## التعديل الثالث

تطوير البرنامج لسحسب فترة زمنية بين شهرين مختلفين وسنتين مختلفتين
